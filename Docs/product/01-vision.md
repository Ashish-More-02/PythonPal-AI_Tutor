# 01 — Vision & Strategy

## The audience

**Absolute-beginner adults with zero coding background.** The mental model is: *"my mom, my
dad, or a random uncle should be able to finish this and write Python on their own."*

Why this audience:
- It's **reachable** by a solo builder — they live in r/learnpython, r/learnprogramming,
  Discord servers, and X/build-in-public. We can put the product in front of 50 of them
  this month for free.
- The pain is **real and recurring** — beginners are overwhelmed, don't know what to do
  next, and quit. That's a problem worth solving.
- We understand it firsthand — we learn better by building with AI guidance than by
  watching tutorials and guessing. This product is the thing we wish we had.

## The problem we solve

A total beginner trying to learn Python today faces three failures:

1. **Tutorial hell.** They watch videos, feel like they're learning, but can't write code
   from a blank file. Passive consumption ≠ ability.
2. **No path.** They don't know what to learn, in what order, or when they're "done."
   Endless choice → paralysis → quitting.
3. **No feedback.** When their code breaks, they're alone. ChatGPT will just hand them the
   answer, so they never actually learn to think.

## The insight

**People learn by doing, with a guide who corrects them in real time.** Writing code
yourself + an AI mentor pointing you in the right direction is a massive speed boost over
watching tutorials and guessing what's next. The whole product is built around that loop.

## Positioning

> **"Go from zero to writing Python on your own — by building, not watching. An AI mentor
> reviews your code and guides you to the next step."**

## The moat — why this isn't "just a GPT wrapper" or "just another course"

We sit between two weak alternatives and beat both for the beginner:

| Alternative | Why it fails the beginner | How we win |
|---|---|---|
| **ChatGPT** | Hands over answers → no learning. Forgets you. No structure. | Socratic review (hints, not answers). Remembers you. A real path. |
| **Static courses** (Codecademy, freeCodeCamp) | Same content for everyone. Canned exercises with fixed answers. No personal feedback on *your* code. | AI reviews *your actual code* and adapts to *your* mistakes. |
| **Replit / raw editor** | A powerful tool, but no guidance. A beginner doesn't know what to build. | Structured syllabus + free practice area, with a mentor either way. |

The defensible core is the **combination**: a real editor + execution + a structured
zero-to-hero syllabus + an AI that reviews your code and remembers your journey. Each piece
exists elsewhere; together, aimed at the true beginner, they're a product.

## Competitive landscape (be honest about it)

- **Codecademy / DataCamp / Mimo** — structured, but static; no personalized code review.
- **Boot.dev / Codédex / Exercism** — project/gamified learning; Exercism has human
  mentors (slow), the others have fixed feedback. None center an adaptive AI mentor.
- **Replit (100 Days of Code)** — great editor + some curriculum; not built around an AI
  that reviews *your* code and adapts.
- **ChatGPT / Claude** — the real competitor for "ask a question," but terrible as a
  structured, memory-having, answer-withholding *teacher*.

**Our wedge:** the personalized *AI code-review loop* tied to a *zero-to-independent path*,
for *true beginners*. That's a specific enough claim to win a niche.

## What success looks like

- **Product:** a beginner finishes the syllabus and writes a small Python program from a
  blank file, unaided.
- **Metric that matters:** Day-2 / Week-2 return rate. Learning is a habit; retention is
  the whole thesis. Everything else is vanity.
- **Resume story:** "Designed and shipped an adaptive learn-by-doing platform with a
  persistent learner model, sandboxed code execution, and an AI code-review loop — used by
  N real learners with X% weekly retention." That is not a GPT wrapper.

## Non-goals (for now)

- Not multi-language. Python only until the loop is proven.
- Not a certification/credential platform. Ability to code is the reward.
